# Files_Spyder
PyQt5项目，从大量csv,xls 和gbc（一种自定义类型）中搜索关键词，并显示在界面中.

基本功能实现。
实现了跳过文件，自定义+ 从配置文件中获取
实现了点击ChecK 单元格 可以查看文件
对界面进行了一定的优化
